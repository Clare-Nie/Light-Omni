# Light-Omni

Light-Omni is a lightweight project homepage and documentation entry point for
omnimemory training dataset construction.

## GitHub Pages

This repository is prepared for project-level GitHub Pages through the
`docs/` directory.

After pushing the repository to GitHub:

1. Open the repository settings.
2. Go to `Pages`.
3. Set source to `Deploy from a branch`.
4. Select branch `main` and folder `/docs`.

The public project page will usually be available at:

```text
https://YOUR_GITHUB_USERNAME.github.io/Light-Omni/
```

## Local Preview

Open `docs/index.html` directly in a browser, or serve the folder with:

```bash
python3 -m http.server 8000 -d docs
```
