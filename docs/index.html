Overview

Project Purpose

Light-Omni is designed as a compact, public-facing companion for a training dataset construction workflow. It focuses on clarity: what the dataset is for, how examples are produced, what metadata is preserved, and how future users can reproduce or extend the pipeline.

This homepage is intentionally static and GitHub Pages friendly. It can be served directly from the repository's docs/ directory without adding a frontend build system.

Dataset Construction Flow
